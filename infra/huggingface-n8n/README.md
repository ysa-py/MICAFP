# TorShield-IR — n8n CI/CD Runner

این Space صفحه‌ی n8n شماست؛ بعد از بالا آمدن، آدرس Space (چیزی شبیه
`https://YOUR_USERNAME-torshield-ir-ci.hf.space`) را مستقیماً در یک تب
جدید مرورگر باز کنید (نه از داخل پیش‌نمایش embed، چون n8n هدر
X-Frame-Options می‌فرستد).

این Space باید **Private** باشد — کد و دیتای .env پروژه حساس است.
